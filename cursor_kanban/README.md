# Kanban Project Manager

A single-board Kanban web app built with Next.js.

## Setup

```bash
cd frontend
npm install
npx playwright install
```

## Development

```bash
cd frontend
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Testing

```bash
cd frontend
npm run test:run    # unit tests
npm run test:e2e    # end-to-end tests
```

## Production

```bash
cd frontend
npm run build
npm run start
```
